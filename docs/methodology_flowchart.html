🔬 Research Methodology

Wildfire Prediction with Pseudo-Absence Generation

Phase 1

📦 Data Collection

Fire occurrence records (GPS ignition points) + environmental covariates
(vegetation, topography, climate) + study area definition

Phase 2

🔧 Data Preprocessing

Spatial filtering · duplicate removal · raster alignment · covariate extraction at fire locations

Decision
Which pseudo-absence strategy?
Strategy A

🎲 Random Placement

Absence points drawn uniformly at random across the full study extent. No environmental constraints.

⚠️ Quality Test (AUC)

Secondary classifier easily separates fire vs. random controls

AUC ≈ 0.90 ✗

🚫 Biased Training Set

Controls ≠ fire-prone conditions. Model learns trivial separation. Predictions are unreliable.

Strategy B

🧠 Heuristic / GAN

GAN trained on fire-location covariate vectors generates environmentally realistic absence candidates.

✅ Quality Test (AUC)

Secondary classifier cannot separate fire vs. GAN controls

AUC ≈ 0.50 ✓

✅ Valid Training Set

Controls ≈ fire-prone conditions. Model learns genuine risk signals. Predictions are ecologically valid.

Phase 3

🤖 Binary Classifier Training

Fire presences + pseudo-absences → feature extraction → train / val / test split → model fitting (RF / XGBoost / DNN) — repeated for each strategy

Phase 4

📊 Model Evaluation

AUC-ROC · Precision / Recall · Feature importance · Spatial validity of fire risk maps — compared across both strategies

Phase 5

🗺️ Fire Risk Prediction Map

Model applied pixel-by-pixel across full study area to produce a continuous fire probability surface [0 – 1]

Conclusion

🏁 Key Finding

GAN/heuristic pseudo-absences produce unbiased training data and significantly more reliable fire risk predictions than random placement.
